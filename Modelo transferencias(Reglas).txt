Condiciones obligatorias para realizar una transferencia:
Cliente tiene que estar verificado (t,f)

Parametros del costo de transferencia 
Si el banco destino es el mismo que el banco del cliente el costo es 0
Si el banco destino es diferente que el banco del cliente el costo es 100
